# TechMC Glossary

> Многоязычный глоссарий технических терминов Minecraft, помогающий игрокам, разработчикам и переводчикам сохранять единообразие терминологии в разных языках.

[Открыть глоссарий онлайн](http://beta.techmc.wiki/glossary) · [Посмотреть исходный CSV](https://github.com/DuskScorpio/TechMC-Glossary/blob/main/TechMC%20Glossary.csv)

</div>

- Упрощённый китайский
- Русский
- Испанский

## Файл глоссария

`TechMC Glossary.csv` содержит следующие столбцы:

| Столбец | Назначение |
|---|---|
| `Category` | Тематическая категория (используется для сортировки) |
| `Short Form` | Распространённое сокращение, если есть |
| `Regex` | Необязательное регулярное выражение для терминов с переменным количеством |
| `Full Form (English)` | Канонический термин на английском |
| `Related` | Ссылки на синонимы и связанные термины |
| `Description` | Определение на английском |
| `<Язык>` / `Description (<Язык>)` | Перевод и локализованное определение |

> [!NOTE]
> Спорные или обсуждаемые записи помечаются символом `*` в конце.

### Соглашения столбца `Related`

Столбец `Related` связывает термины с помощью типизированных префиксов:

| Префикс | Значение | Пример |
|---|---|---|
| `synonym:` | Та же концепция, взаимозаменяемое название | `synonym:Instant Tile Tick` |
| `see:` | Другая, но связанная концепция | `see:Comparator Update Detector` |

Несколько записей разделяются `; `, например:

```
synonym:Bounding Box; synonym:Hitbox; see:Collision Box
```

## Как внести вклад

> [!TIP]
> Для большинства правок **[веб-редактор на beta.techmc.wiki/glossary](http://beta.techmc.wiki/glossary)** — самый удобный вариант и рекомендуется к использованию. Для сложных или крупных изменений по-прежнему рекомендуется работать локально с `git` и клонированным репозиторием.

Если вы предпочитаете работать с файлом напрямую:

- Откройте [issue](https://github.com/DuskScorpio/TechMC-Glossary/issues), чтобы обсудить изменение или сообщить о проблеме.
- Сделайте форк репозитория, отредактируйте `TechMC Glossary.csv` и отправьте pull request.
- Сохраняйте единое форматирование и поясняйте свои изменения в описании PR.

### Ручное редактирование CSV

1. [Скачайте `TechMC Glossary.csv`](https://github.com/DuskScorpio/TechMC-Glossary/raw/main/TechMC%20Glossary.csv) (правый клик → Сохранить как).
2. Откройте файл в Microsoft Excel, LibreOffice Calc или другом табличном редакторе.
3. Убедитесь, что строки отсортированы по столбцу `Category` по алфавиту.
4. Сохраните как **CSV UTF-8 (разделитель — запятая) (`*.csv`)**.
5. Отправьте изменения через pull request.

> [!IMPORTANT]
> Всегда сохраняйте файл в кодировке **UTF-8**, чтобы избежать проблем с нелатинскими символами.
